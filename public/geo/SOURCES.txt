punjab_divisions.geojson
------------------------
Derived from Pakistan ADM2 (admin level 2 = Division) boundaries.

Upstream repository: https://github.com/PakData/GISData
Source file: PAK-GeoJSON/PAK_adm2.json (same data as GADM / Global Administrative Areas).

This extract includes only features where NAME_1 = "Punjab" (Punjab province, Pakistan).
Each feature.properties.division matches the Wikipedia-style name without the word "Division"
(e.g. "Lahore", "Dera Ghazi Khan").

Coverage note (important)
-------------------------
The PakData/GISData ADM2 layer reflects an older GADM snapshot: it contains 8 division
polygons for Punjab. **Sahiwal** is not a separate ADM2 polygon there; this repo’s
`punjab_divisions.geojson` adds Sahiwal by merging the relevant ADM3 districts
(Sahiwal, Pakpattan, Okara, Okara 1) from `PAK_adm3.json` (see
`scripts/build-punjab-divisions-geojson.mjs`). **Gujrat** is still not a separate polygon
in this ADM2 extract; for a full 10-division map, merge ADM3 for Gujrat, use a newer
release, or update polygons manually.
